New Arrivals

PlayStation 5

PlayStation 5 (PS5) is a video game console developed by Sony Interactive Entertainment. It features a custom AMD Zen 2-based CPU, a custom RDNA 2-based GPU, and ultra-fast solid-state drive (SSD). It also has ray tracing capabilities and 8K graphics support. It was released in November 2020.

Learn More

Iphone 13

The iPhone 13 is a line of smartphones developed and marketed by Apple Inc. It features A15 Bionic chip, 5G connectivity, Ceramic Shield front cover, ProMotion display with ProMotion technology, and improved camera system with Night mode and Deep Fusion. It also has a new design with a smaller notch and Face ID that works at different angles.

Learn More

Eero Home Wifi

Eero is a mesh WiFi system that provides a fast and reliable wireless network throughout a home. It uses multiple devices, called eero, that work together to blanket a home in WiFi. It has a simple set up and easy to manage with mobile app and it also comes with advanced security features to protect your network and devices. It also supports Alexa and Google assistant.

Learn More

Alexa Echo

Alexa Echo is a voice-controlled smart speaker developed by Amazon. It can play music, control smart home devices, make calls, and answer questions with the help of Amazon's virtual assistant, Alexa. It is available in different models with different features and design. It can also connect to other devices and be integrated with a wide range of smart home systems.
